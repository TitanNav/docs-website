## 構文

```
setApplicationBuild:(NSString *_Nonnull)versionString;
NewRelic.setApplicationBuild:(NSString*)versionString;
```

New Relic にレポートされるアプリケーションのビルド番号を設定します。

## 要件

バージョン 5.1.0New Relic iOS SDK の以降。

## 説明

デフォルトでは、New Relic はビルド番号を報告するときに`CFBundleVersion`を使用します。ただし、このメソッドを呼び出すことで、報告されたビルド番号をオーバーライドできます ( `startWithApplicationToken`を呼び出す**前に**)。
