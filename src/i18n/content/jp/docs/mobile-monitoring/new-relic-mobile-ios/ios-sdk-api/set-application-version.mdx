## 構文

```
setApplicationVersion:(NSString *_Nonnull)versionString;
NewRelic.setApplicationVersion:(NSString*)versionString;
```

New Relic にレポートされるアプリケーションのバージョンを設定します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

デフォルトでは、New Relic はビルド番号を報告するときに`CFBundleShortVersionString`を使用します。ただし、このメソッドを呼び出すことで、報告されたビルド番号をオーバーライドできます ( `startWithApplicationToken`を呼び出す**前に**)。

**例：**

```

```
