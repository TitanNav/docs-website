APMでは、 **タグ** に [キートランザクション](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events) を割り当てることができます。

タグは、トランザクションに関連付けられたメタデータです。共通点のあるトランザクションに適用する場合に便利です。たとえば、チェックアウト プロセスの一部であるすべての主要なトランザクションに`checkout`タグを適用できます。次に、 `checkout`でフィルター処理して、これらのすべてのトランザクションを 1 か所で確認できます。

## タグの割り当て [#tagging_assign]

キー・トランザクションにタグを割り当てるには

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > All capabilities > Key transactions**に移動します。

2. インデックスページから、タグ付けしたいトランザクションの名前の横にあるtag <Icon name="fe-tag"/>アイコンを選択します。

3. タグ名ボックスに、タグの名前を入力します。スペースキーを押すと、タグが適用されます。

   <Callout variant="tip">
     コンマ`,` 、山かっこ`< >` 、かっこ`( )` 、または角かっこ`[ ]`を除く、任意の Unicode 文字を使用します。
   </Callout>

4. タグ名の欄外を選択すると、タグの作成を終了します。

## 複数トランザクションのタグ付け [#tagging_multiple]

インデックスに表示されているすべてのトランザクションにタグを割り当てることができます。

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > All capabilities > Key transactions**に移動します。
2. [タグの割り当て](#tagging_assign) のタグ作成手順に従ってください。

バルクタグを削除するには

* バルクタグのドロップダウンから、タグ名の **x** シンボルを選択します。

## フィルター [#tagging_filtering]

キー・トランザクション・インデックスをタグでフィルタリングするには

1. キー・トランザクション・インデックス検索フィールドで、

   <Icon name="fe-chevron-down"/>

   ドロップダウン・アイコンを選択し、 **by tags** を選択します。

2. タグ名を入力してください。
