## 통사론

```
+ (void) stopCurrentInteraction:(NSString*)interactionIdentifier;
```

제공된 `interactionIdentifier` 과 연결된 상호작용 추적을 중지합니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

이 메서드는 `interactionIdentifier` ( `startInteractionWithName: `메서드에 의해 반환됨)와 연결된 상호작용 추적을 중지합니다. 상호 작용 추적을 완료하기 위해 이 메서드를 호출할 필요는 없습니다(상호 작용 추적은 자체적으로 지능적으로 완료됨). 그러나 보다 불연속적인 상호 작용 기간을 원하는 경우 이 방법을 사용하십시오. 이 API를 사용하는 방법에 대한 컨텍스트는 상호 작용 추적에 대한 설명서를 참조하세요.

* [오브젝티브-C](/docs/mobile-monitoring/new-relic-mobile-ios/ios-sdk-api/ios-sdk-api-guide#objc-interactions)
* [빠른](/docs/mobile-monitoring/new-relic-mobile-ios/ios-sdk-api/ios-sdk-api-guide#swift-interactions)

## 예

### 오브젝티브-C [#obj-c]

**예시:**

```
NSString *identifier = [NewRelic startInteractionWithName: @"myInteractionName"];
[NewRelic stopCurrentInteraction: identifier];
```

### 빠른

**예시:**

```
let identifier = NewRelic.startInteraction(withName: "myInteractionName")
NewRelic.stopCurrentInteraction(identifier)
```
