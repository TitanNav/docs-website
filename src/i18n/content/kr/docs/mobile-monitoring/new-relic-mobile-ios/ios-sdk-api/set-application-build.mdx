## 통사론

```
setApplicationBuild:(NSString *_Nonnull)versionString;
NewRelic.setApplicationBuild:(NSString*)versionString;
```

New Relic에 보고된 애플리케이션의 빌드 번호를 설정합니다.

## 요구 사항

버전 5.1.0 또는 이후 버전의 New Relic iOS SDK.

## 설명

기본적으로 New Relic은 빌드 번호를 보고할 때 `CFBundleVersion` 을 사용합니다. 그러나 이 메서드를 호출하여 보고된 빌드 번호를 재정의할 수 있습니다( `startWithApplicationToken` 호출 **전** ).
