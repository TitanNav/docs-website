## 통사론

```
setApplicationVersion:(NSString *_Nonnull)versionString;
NewRelic.setApplicationVersion:(NSString*)versionString;
```

New Relic에 보고된 애플리케이션의 버전을 설정합니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

기본적으로 New Relic은 빌드 번호를 보고할 때 `CFBundleShortVersionString` 을 사용합니다. 그러나 이 메서드를 호출하여 보고된 빌드 번호를 재정의할 수 있습니다( `startWithApplicationToken` 호출 **전** ).

## 예

### 오브젝티브-C [#obj-c]

**예시:**

```
[NewRelic setApplicationVersion:@"1.2.3"];
```

### 빠른 [#swift]

**예시:**

```
NewRelic.setApplicationVersion("1.2.3")
```
